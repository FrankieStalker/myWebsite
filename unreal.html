Shopkeepers Dungeon

This was a group project where me and 3 other students had to create a game. We opted for a dungeon crawler, RPG where the player would enter a shop only to be sprung into a cave having to fight off skeletons and obtain magical spells along the way. For this project I was tasked with creating environments and UI for the game. I had no prior experience of working with UE4 so I did learn a lot from this project.
